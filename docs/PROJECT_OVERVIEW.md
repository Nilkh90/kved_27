## Название проекта
**NACE 2.1-UA** — сервис перехода украинского бизнеса с КВЭД-2010 на NACE 2.1-UA (вступает в силу в 2027 году, Приказ Госстата № 191).

## Технологический стек
**Laravel 11 + Livewire 3 + PostgreSQL + Redis**

Выбор обоснован:
- PHP знает большинство украинских backend-разработчиков → дешевле нанимать
- Laravel из коробки даёт всё необходимое: роутинг, ORM, очереди, API-токены, кеш
- Livewire 3 обеспечивает реактивный поиск без написания отдельного JS-фреймворка
- Shared хостинг или простой VPS от $5-10/мес — нет зависимости от Vercel/Neon
- Laravel Sanctum — готовые API-токены для монетизации на фазе 2

## Проблема
С 2027 года Украина переходит на новый классификатор NACE 2.1-UA. Для бизнеса это означает:
- Неопределённость: «изменился ли мой код?»
- Риск штрафов за несвоевременную перерегистрацию
- Отсутствие понятного официального интерфейса для поиска соответствий

## Решение
Веб-приложение, которое:
1. Находит соответствие между старым кодом КВЭД-2010 и новым NACE 2.1-UA за 10 секунд
2. Объясняет юридические последствия (нужна перерегистрация или нет)
3. Представляет изменения понятным языком без бюрократического жаргона

## Целевая аудитория
| Сегмент | Потребность |
|---|---|
| Бухгалтеры и аутсорс-компании | Массовая проверка кодов клиентов, API-доступ |
| ФОП / ФЛП | Быстро понять: «что делать моему бизнесу?» |
| Руководители МСБ | Оценить риски, подготовить документы |
| Юристы и регистраторы | Справочный инструмент со ссылками на НПА |

## Монетизация (фаза 2+)
- Платное API для интеграции с 1С, Checkbox, Taxer, Liga:ZAKON (Laravel Sanctum)
- AI-помощник в личном кабинете (подбор кода по описанию бизнеса)
- B2B подписки для аутсорс-бухгалтерий
